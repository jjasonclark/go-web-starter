# go-web-starter
Starter project for Go (Golang) backed web sites

## Build commands
* `npm run build`

## Dependencies
* `go get -u github.com/jteeuwen/go-bindata/...`
* `go get -u github.com/elazarl/go-bindata-assetfs/...`
